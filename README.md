# zig-0-16-migration

A Kimi skill for Zig 0.16.0 migration.

## What's inside

- `zig-0-16-migration/` — Chinese version of the skill
- `zig-0-16-migration-en/` — English version of the skill
- `zig-0-16-migration.skill` — Packaged Chinese skill (zip)
- `zig-0-16-migration-en.skill` — Packaged English skill (zip)

## Usage

For [Kimi Code CLI](https://kimi.moonshot.cn/) users, place the `.skill` file into your skills directory, or unpack the directory directly into `~/.config/agents/skills/`.
